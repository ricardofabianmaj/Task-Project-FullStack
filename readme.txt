Para iniciar esse projeto você deve configurar o MySQL com a sua conta no arquivo './backend/connection.js'

Além disso, abra dois terminais, um para o backend e outro para o frontend

Após isso, configure cada terminal para executar sua devida aplicação

Usando o do backend da seguinte maneira:

cd backend
node main.js

E o frontend da seguinte maneira:

cd frontend
npm start

Verifique se você possui todas as dependencias usadas no projeto...

Importante lembrar que está ainda é a primeira versão da aplicação, logo, oque o sistema funciona bem porém atualizações serão feitas futuramente, como melhora na lógica do aplicativo, design, upload e até mesmo o repositório do mesmo...
